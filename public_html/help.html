PokeMeow

Gamepage About Account Logout

PokeMeow is a simplified, command-line version of Pokemon. It is based on theDiscord bot “PokeMeow,” which allows users to catch (and soon, battle) Pokemonwithin a chat window.

Main Commands:
  • random-encounter - starts an encounter with a random pokemon
  • view-party - prints out a list of pokemon in your party
  • view-pokemon - prints out a list of the pokemon you've caught
  • view name - prints out more information about pokemon 'name'
  • remove name - removes pokemon 'name' from your party
  • add name - adds pokemon 'name' to your party
  • release name - releases the pokemon 'name' back to the wild
Random Encounter Commands:
  • throw-ball - throws a ball to try and catch a pokemon
  • run - run from the random encounter
